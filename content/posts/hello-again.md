I've rebuilt this site from scratch. New foundation, clean slate.

## Why bother

The old version had quietly rotted: a half-configured setup, no posts, and
a build I no longer trusted. Rather than patch it, I started over with
something I can actually maintain.

This time the goals are modest:

- **Write more, fuss less.** A post should be one Markdown file.
- **Stay fast.** Static HTML, no client-side bloat.
- **Own the words.** Everything lives in a Git repo I control.

## How it's built

The site runs on [Hugo](https://gohugo.io/) with the
[PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme — light and
dark modes included — and deploys to GitHub Pages automatically on every push
to `main`.

That's it. No database, no comments to moderate, nothing to babysit.

## What's next

I want to use this space for working notes and longer essays alike. If you're
reading this, the experiment has at least started. Let's see where it goes.
